# 6DPoseAnnotator

An interactive 6 degree-of-freedom pose annotation tool using point cloud processings.

<img src="./example.png" width="5000px">

## Requirements
- [open3d==0.11.1](http://www.open3d.org/)
- [opencv-python==4.4.0.44](https://opencv.org/)
